# tjc
Textarea JSON Converter

##cモード(会話まとめ)  
tlc c [タイトル区切り文字] [投稿区切り文字] [うた、えりんぎ、しろる、紅茶それぞれの識別文字 ,区切り]

##kモード(発言まとめ)  
tlc k [区切り文字] [うた、えりんぎ、しろる、紅茶それぞれの判別文字 ,区切り] [誤字、おつうわ、チャット、ツイートそれぞれの識別文字 ,区切り]

改行文字で要素を区切る  
引数の区切りは半角スペース  
終了文字はセミコロン

#例

tjc c @ # (u,k,r,t)  
@靴履けよ@220205  
t#👀  
r#🍫  
r#👀&lt;br&gt;👃&lt;br&gt;👄  
t#🦵🦵  
u#🦵🦵  
t#🩴🩴  
r#靴履けよ  
u#2足分足りないですよ  
r#👠👠
